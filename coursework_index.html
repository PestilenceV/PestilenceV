Курсовая работа

Тема: Разработка игрового приложения "Электрические схемы" на JS.

Цель и задачи

Цель: Создать JS-приложение для изучения электрических схем. Задачи: Определить элементы (реле, лампочки, резисторы), подобрать формулы, собрать схемы. Идеи: Собрать цепь (эффект при успехе, ограниченные попытки), расставить светодиоды, расчёт по формулам.

Ход работы

Модуль Game_LastName_GameTitle. Миниавторизация (имя). Рейтинг. 3 уровня (3-5 вопросов каждый, генерация). События: Перемещение, клик, наведение. Время, ускорение, баллы (штрафы). Динамика элементов, анимация. LocalStorage для данных. 3 страницы: Заставка, игра, рейтинг. На хостинге.

Источники

JS-документация, ресурсы по электрике (формулы, схемы). Графические библиотеки (если не влияют на локальную работу).
